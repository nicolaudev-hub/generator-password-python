# Password Generator Python

Projeto desenvolvido em Python com o objetivo de praticar lógica de programação e conceitos básicos de segurança cibernética.

## Funcionalidades atuais

- Geração de senhas aleatórias
- Definição do tamanho da senha pelo usuário
- Utilização de letras, números e caracteres especiais

## Tecnologias utilizadas

- Python 3
- Biblioteca random
- Biblioteca string

## Objetivos de aprendizado

Este projeto está sendo desenvolvido de forma incremental para praticar:

- Variáveis
- Strings
- Estruturas condicionais (if/else)
- Laços de repetição (for)
- Listas
- Funções
- Validação de entrada
